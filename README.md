# Giftly 

Design personalized gift cards with festive themes, custom messages, and share them with your loved ones.

ðŸ”— **Live Demo:** [https://giftly-lovat.vercel.app/gift-card](https://giftly-lovat.vercel.app/gift-card)

## Features

-  Themes & Backgrounds: gradient themes with festive stickers and Custom background upload
-  Personalization: Custom recipient name, Sender name, Editable heading & tagline, Quick Message themes
-  Card Styling: Border styles, Custom border color, Corner decorations, Text color & size options
-  Photo Support: Upload personal photos to include on cards
-  Export & Share: Generate as high-quality PNG, Share to Telegram and Messenger as well as Direct download
-  Language: Bilingual interface (English / Khmer)
-  Real-time Preview: Live card preview while editing

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- MongoDB Atlas
- TypeScript
