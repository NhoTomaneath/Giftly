# Giftly 

A simple New Year gift card generator built with Next.js.

ðŸ”— **Live Demo:** [https://giftly-lovat.vercel.app/gift-card](https://giftly-lovat.vercel.app/gift-card)

## Features

-  Create personalized gift cards with stickers
-  Add custom recipient name and message
-  Real-time card preview
-  Export as PNG image
-  Bilingual support (English/Khmer)

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- MongoDB Atlas
- TypeScript
